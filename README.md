# C Star Generator

Simple CLI app for generate shape using character with custom size

## Run Locally

Clone the project

~~~bash
  git clone https://github.com/Sleepy4k/c-star-generator.git
~~~

Go to the project directory

~~~bash
  cd c-star-generator
~~~

Build the c file

~~~bash
  gcc ./src/main.c -o ./output/main
~~~

Start the project

~~~bash
  ./output/main
~~~
